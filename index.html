    Unicode 编码&解码

    • Unicode在HTML中格式
    • Unicode在JS中格式
    • Unicode在CSS中格式

    • HTML实体字符串（16进制）

      <!--应用在HTML中示例-->
<div>&#x7F16;&#x7801;&#x89E3;&#x7801;</div>

    • HTML实体字符串（10进制）

      <!--应用在HTML中示例-->
<div>&#32534;&#30721;&#35299;&#30721;</div>

    • 16进制Unicode编码（可以同时用于HTML与JS中）

      <!--应用在HTML中示例-->
<div>\u7F16\u7801\u89E3\u7801</div>
<script>
  // 应用在JavaScript中示例
  alert('\u7F16\u7801\u89E3\u7801');
</script>

    • 用于CSS中16进制Unicode编码格式

      /*
* 应用在CSS中示例
*/
<style type="text/css">
.box{
  &::after{
    content:'\7F16\7801\89E3\7801';
  }
}
</style>
